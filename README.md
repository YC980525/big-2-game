# Big 2 Game

## Overview
A simple terminal-based Big 2 game crafted in JAVA.

## Notes
1. Card suits follow the order: ♣ < ♦ < ♥ < ♠.

2. Supported play patterns include:
   - Single
   - Pair
   - Straight
   - Full House

## How to Play
To begin, execute the following command:
```sh
bash script/run.sh
```
